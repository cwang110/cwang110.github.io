007-Go语言多个变量同事赋值

1.交换变量的常见算法需要一个中间变量进行变量的临时保存。用传统方法编写变量交换代码如下

var a int = 100
var b int = 200
var t int
t = a
a = b
b = t
fmt.Println(a, b)

2.使用算法来交换

var a int = 100
var b int = 200
a = a ^ b
b = b ^ a
a = a ^ b
fmt.Println(a, b)

3.使用Go语言多重赋值特性

var a int = 100
var b int = 200
b, a = a, b
fmt.Println(a, b)

4.使用Go语言进行排序时就需要使用交换，代码如下

type IntSlice []int

func (p IntSlice) Len() int           { return len(p) }
func (p IntSlice) Less(i, j int) bool { return p[i] < p[j] }
func (p IntSlice) Swap(i, j int)      { p[i], p[j] = p[j], p[i] }

代码说明如下：

  • 第 1 行，将 IntSlice 声明为 []int 类型。
  • 第 3 行，为 IntSlice 类型编写一个 Len 方法，提供切片的长度。
  • 第 4 行，根据提供的 i、j 元素索引，获取元素后进行比较，返回比较结果。
  • 第 5 行，根据提供的 i、j 元素索引，交换两个元素的值。
